# Python Chatbot Projects

This repository contains multiple chatbot projects created using Python and Tkinter.

The projects were built to learn:
- GUI development
- User interaction
- Chatbot logic
- Natural Language Processing (NLP)
- Machine Learning basics
- Streamlit web applications
- Session state handling
- Web-based GUI development
  
The repository includes beginner to intermediate chatbot applications, starting from simple rule-based responses to a Machine Learning based chatbot.

## Projects Included

# Simple Chatbot

A beginner rule-based chatbot created using Python and Tkinter.

The chatbot responds to predefined user questions and simple conversational messages related to Python programming and general topics.

### Features

- GUI based chatbot
- Predefined responses
- Interactive conversation window
- Python related questions and answers
- User input handling

### Technologies Used

- Python
- Tkinter

# Streamlit Chatbot

This project is a simple chatbot web application created using Python and Streamlit.

The chatbot responds to predefined user questions related to:
- Python programming
- Basic coding concepts
- General conversation

The chatbot stores conversation history during the session and displays responses in an interactive chat format.

## Features

- Web-based chatbot interface
- Interactive user input
- Chat history using session state
- Rule-based chatbot responses
- Python programming related questions
- Beginner-friendly chatbot system

## Technologies Used

- Python
- Streamlit

# Quiz Bot

A GUI based quiz chatbot that asks multiple-choice questions and calculates the final score.

The bot allows users to select answers using radio buttons and displays whether the selected answer is correct or wrong.

### Features

- Multiple-choice quiz system
- Score calculation
- Interactive GUI
- Real-time answer checking
- Quiz result display

### Technologies Used

- Python
- Tkinter

# ML Chatbot

A Machine Learning based chatbot created using Python, Tkinter, NLTK, and Scikit-learn.

The chatbot uses NLP techniques and a trained intent dataset to predict suitable responses for user input.

The project demonstrates concepts such as:
- Natural Language Processing
- Text vectorization
- Machine Learning classification
- GUI integration

### Features

- GUI based chatbot
- Intent recognition
- NLP text processing
- TF-IDF vectorization
- Logistic Regression model
- JSON based training data
- Real-time chatbot responses

### Technologies Used

- Python
- Tkinter
- NLTK
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression

## How to Run

Run any Python file individually:

python filename.py

Example:

python Chatbot.py

Run any Python file involving streamlit:

 Install Streamlit:

pip install streamlit

  Run the chatbot application:

streamlit run chatbot_streamlit.py

After running the command, Streamlit will open the chatbot in your web browser.
